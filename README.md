### Car Racing Game in C

This project is a simple **console-based Car Racing Game** developed in **C** for Windows. The player controls a car (`A`) using the **A** and **D** keys to move left and right while avoiding falling obstacles (`X`). The game uses real-time keyboard input, random obstacle generation, collision detection, and score tracking. As the score increases, the game speed gradually increases, making it more challenging. The game ends when the player's car collides with an obstacle.

**Features**

* Console-based gameplay
* Real-time keyboard controls
* Random obstacle generation
* Collision detection
* Dynamic score counter
* Increasing game difficulty
* Quit option using the **Q** key

**Technologies Used**

* C Programming
* Standard C Libraries (`stdio.h`, `stdlib.h`, `time.h`)
* Windows API (`windows.h`, `conio.h`)

This project demonstrates the use of **structures, arrays, loops, functions, random number generation, and basic game logic** in C programming.
